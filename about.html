Quem somos?

O AVA - Ambiente Virtual de Aprendizagem - é uma plataforma de estudo das principais ferramentas de desenvolvimento de páginas web para aqueles que não sabem por onde começar.


A plataforma conta com aulas do YouTube para os temas Lógica de Programação, HTML5, CSS3 e Javascript, pilares básicos para a construção de um website. A seleção das aulas leva em consideração o domínio e a praticidade da explicação dos professores sobre esses conceitos e ferramentas.


O AVA não é responsável pelo conteúdo das aulas. Todos os direitos estão reservados aos criadores de conteúdo do YouTube, no caso, os professores.


Este site é um projeto desenvolvido por um aluno da disciplina "Desenvolvimento Web", do Instituto Federal de Tecnologia de São Paulo, utilizando os principais conceitos de HTML, CSS e Javascript.

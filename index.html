Lógica de Programação

Lógica de programação é o modo como se escreve um programa de computador, um algoritmo. Um algoritmo é uma sequência de passos para se executar uma função. Dominar a lógica de programação é essencial para a construção de códigos em qualquer linguagem.

HTML

HTML significa Hypertext Markup Language. Ele nos permite criar e estruturar seções, parágrafos, cabeçalhos e links para páginas da internet ou aplicações. O HTML é o esqueleto de uma página web, fundamental para a organização de um site.

CSS

CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é uma linguagem de estilização de páginas web, usada para descrever a apresentação de um documento escrito em HTML. O CSS define como os elementos de um site serão apresentados, como seu tamanho, fonte, cores e etc.

Javascript

Javascript é a linguagem de programação mais utilizada na criação de páginas web, interagindo de forma fácil com o HTML e o CSS. Possui diversos frameworks que agilizam a sua utilização, se tornando uma linguagem extremamente popular.
